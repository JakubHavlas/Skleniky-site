# Skleniky-site

# Claude.md – Landing Page Specification
## Project: Smart Growing Cabinet for Restaurants

---

# Objective

The purpose of this website is **NOT** to sell a product.

The purpose is to validate market demand.

Primary KPI:

> Number of restaurants applying for the Pilot Program.

Everything on this page should move the visitor toward one action:

**Apply to become a pilot restaurant.**

---

# Brand Positioning

We are **not** selling a greenhouse.

We are selling:

> Fresh herbs, always available, with almost zero effort.

The website should communicate:

- Premium
- Modern
- Trustworthy
- Innovative
- Minimalistic

Never feel like an agricultural company.

Instead, feel closer to:

- Apple
- Stripe
- Linear
- Vercel
- Tesla
- Nothing

---

# Target Audience

Primary audience:

- Restaurant owners
- Executive chefs
- Fine dining restaurants
- Hotels
- Modern bistros
- Premium cafés

These people are busy.

The website must explain everything within 30 seconds.

---

# Design Language

## General Style

Minimalistic.

Large typography.

Large white spaces.

High-end product presentation.

No unnecessary graphics.

No stock photos.

No green leaves everywhere.

No "eco startup" feeling.

The cabinet itself should feel like a premium appliance.

# Copywriting Rules

Never talk about technology first.

Talk about outcomes.

Good

Fresh herbs every day.

Bad

AI-powered hydroponic system.

---

Good

Harvest basil before serving.

Bad

Advanced growing algorithms.

---

Good

Almost zero maintenance.

Bad

IoT-enabled nutrient optimization.

---

# Images

Avoid stock photography.

Instead use

- Product renders
- Kitchen renders
- Macro herb shots
- Premium restaurant interiors

Everything should feel realistic.

---

# Motion

Very subtle.

Allowed

Fade

Slide

Hover

Small floating effect

Not allowed

Large animations

Parallax

Flying leaves

Excessive motion
